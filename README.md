# Repository with Node.js code examples
***
## fs (File System)
***
### [dir-report](./fs/dirs/dir-report)

A tool for analyzing directory content and managing reports.

- **Usage**: `node fs/dirs/dir-report/dir-report.js <action> <folderPath>`
- **Available actions**:
  - `report` — generate a report of the folder content.
  - `print` — output an existing report.

### [extensions-scanning](./fs/dirs/extensions-scanning)

A script that scans a specified directory and counts files grouped by their extensions.

- **Usage**: `node fs/dirs/extensions-scanning/ext-scan.js <folderPath>`
